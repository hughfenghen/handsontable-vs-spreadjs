    Configure Themes
    In This Topic

    You can specify themes for cells, columns, rows, or the widget.

    You can use the themeFont method for the cell, row, or column. You can use the currentTheme method for the sheet.

    The following theme names can be used with the currentTheme method:

    • Default
    • Office
    • Apex
    • Aspect
    • Concourse
    • Civic
    • Oriel
    • Origin
    • Paper
    • Solstice
    • Technic
    • Trek
    • Urban
    • Verve
    • Equity
    • Flow
    • Foundry
    • Median
    • Metro
    • Module
    • Opulent

    The following image displays the accents that you can set for the theme colors.

    Using Code

    This example sets the theme with the currentTheme method.

    JavaScript
    Copy Code
    activeSheet.currentTheme("Metro");

    Using Code

    This example sets theme colors.

    JavaScript
    Copy Code

    activeSheet.setColumnWidth(0, 22);
    activeSheet.setColumnWidth(1, 30);
    activeSheet.setColumnWidth(2, 22);
    activeSheet.getCell(1, 3, GC.Spread.Sheets.SheetArea.viewport).value("Theme Colors").themeFont("headings").hAlign(GC.Spread.Sheets.HorizontalAlign.center);
    activeSheet.addSpan(1, 3, 1, 10);
